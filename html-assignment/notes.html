5 Tags I Have Learnt in Class

img
This tag is used to display an image on the web page.
<a>
This tag creates a clickable link to another page or website.
<form>
This tag creates a form to collect information from the user, like name or email addresses.
<table>
This tag is used to make a table that can dispaly data in rows and columns.
<!DOCTYPE html>
This is not a tag, but a declaration that tells the browser to use HTML5.
